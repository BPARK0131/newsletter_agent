# 📡 IETF Standardization Radar

IETF Datatracker 기반 **표준화 컨텍스트**입니다. HITL Review에서 기사 중요도를 판단할 때 참고하고, 뉴스레터 하단 **표준화 레이더** 섹션으로 활용합니다.

> ⚠️ 일반 뉴스 기사가 아닙니다. 승인/반려 대상이 아닙니다.

## IDR / BGP

- **연결 카테고리:** Routing/Internet Operations, IP Security
- **주요 키워드:** BGP, Route Leak, BGP Hijack, Routing Policy
- **기술 영역:** BGP
- **의미:** BGP 확장·정책·Route Leak·Routing Security 논의 추적

**최신 참고 문서:** BGP UPDATE for SD-WAN Edge Discovery (WG draft, 2026-06-16)
- `draft-ietf-idr-sdwan-edge-discovery`
- https://datatracker.ietf.org/doc/draft-ietf-idr-sdwan-edge-discovery/

## BESS / EVPN

- **연결 카테고리:** DataCenter Network, Backbone/Backhaul
- **주요 키워드:** EVPN, VXLAN, VPN
- **기술 영역:** EVPN/VPN
- **의미:** DC Fabric 및 VPN 서비스 구현 방향과 연결

**최신 참고 문서:** Integrated Routing and Bridging in Ethernet VPN (EVPN) (RFC, 2026-05-20)
- `rfc9135`
- https://datatracker.ietf.org/doc/rfc9135/

## LSR / IGP

- **연결 카테고리:** Routing/Internet Operations, Backbone/Backhaul
- **주요 키워드:** IS-IS, OSPF, IGP, Link State
- **기술 영역:** IS-IS/OSPF
- **의미:** Link State 라우팅·IGP 확장 동향과 연결

**최신 참고 문서:** Dynamic Flooding on Dense Graphs (WG draft, 2024-10-31)
- `draft-ietf-lsr-dynamic-flooding`
- https://datatracker.ietf.org/doc/draft-ietf-lsr-dynamic-flooding/

## SPRING / Segment Routing

- **연결 카테고리:** Backbone/Backhaul, Transport/DCI
- **주요 키워드:** SR-MPLS, SRv6, Segment Routing
- **기술 영역:** Segment Routing
- **의미:** SR-MPLS/SRv6 및 백본·기간망 아키텍처 전환 흐름

**최신 참고 문서:** Path Segment Identifier in MPLS-Based Segment Routing Networks (RFC, 2026-05-20)
- `rfc9545`
- https://datatracker.ietf.org/doc/rfc9545/
