# 요약

Vultr에서 RoCEv2 기반의 대규모 RDMA 이더넷 패브릭을 자동화하고 운영할 NetDevOps 엔지니어를 모집합니다.
Ansible과 Python 기반 프레임워크로 언더레이·오버레이 네트워크의 프로비저닝, 검증, 복구를 자동화하며 NetBox/OpsMill 등 소스 오브트루스와 연동합니다.
gNMI, Prometheus, Kafka 등으로 텔레메트리 수집·상관관계 파이프라인을 구축해 실시간 네트워크 상태와 성능을 모니터링하고 CI/CD 워크플로로 구성 변경의 검증 및 롤백을 구현합니다.

# 핵심 포인트

- RoCEv2/RDMA 패브릭 성능 최적화와 ECN/PFC, 패스 대칭성 조정 업무
- Ansible, Python 중심의 자동화 프레임워크 개발 및 운영
- NetBox/OpsMill 등 소스 오브트루스와의 통합을 통한 의도 기반 구성·검증
- gNMI, Prometheus, Kafka 등 텔레메트리 스택을 활용한 실시간 모니터링 파이프라인
- CI/CD(GitHub Actions/Jenkins/ArgoCD 등)를 통한 네트워크 구성 변경의 검증과 롤백 구현
