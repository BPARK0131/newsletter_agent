# 요약

자연어 요청을 코드 에이전트가 InfluxDB 쿼리로 변환해 텔레메트리 기반 인터페이스 상태 보고서를 생성하고, 실시간 네트워크 토폴로지를 문서화된 설계와 비교해 구성 편차를 자동 식별하는 시연을 다룹니다.
AI 에이전트, 텔레메트리, 문서화(NetBox), 시뮬레이션(GNS3)과 실장비(예: SONiC)의 결합을 통해 디지털 트윈 기반의 지능형 네트워크 관리 가능성을 보여줍니다.
InfluxDB와 같은 시계열 DB를 쿼리해 상태·성능 데이터를 추출하고 자동화된 리포트와 편차 탐지에 활용하는 워크플로를 강조합니다.
관련 데모와 강의는 'Infrastructure as Prompt' 마스터클래스에서 자세히 소개됩니다.

# 핵심 포인트

- 자연어→InfluxDB 쿼리 변환 에이전트로 텔레메트리 질의 자동화 가능
- 인터페이스 상태 리포트 생성 및 실시간 토폴로지와 문서 설계 간 자동 비교로 구성 편차 탐지
- 디지털 트윈(시뮬레이션)과 실장비 연동으로 설계 검증·운영 테스트 통합
- 사용 기술 스택 예: InfluxDB, Telemetry, GNS3, NetBox, SONiC, AI 에이전트
