# 요약

이 글은 Meraki와 Catalyst Center를 함께 운영하는 하이브리드 네트워크에서 분산된 운영 가시성을 하나의 화면으로 통합하는 방법을 설명한다.
Global Overview를 통해 검색, 알림 상관분석, 문제 우선순위 지정, 트러블슈팅을 더 빠르게 수행하는 것을 목표로 한다.
핵심은 통합된 뷰를 운영 시작점으로 삼고, 하이브리드 환경에 맞는 운영 모델을 선택해 확장성을 높이는 것이다.
또한 Cisco Cloud Control과의 연계를 통해 네트워킹, 보안, 컴퓨트, 옵저버빌리티, 협업을 아우르는 방향을 제시한다.

# 핵심 포인트

- Meraki와 Catalyst Center의 운영 정보를 Global Overview로 통합해 분산된 가시성을 줄인다.
- 여러 대시보드를 오가며 알림을 맞춰보는 대신, 한 곳에서 검색과 상관분석을 수행하도록 설계됐다.
- 문제 우선순위 지정과 빠른 트러블슈팅을 하이브리드 운영의 주요 사용 사례로 제시한다.
- 대기업과 MSP처럼 다양한 사이트를 관리하는 환경에서 운영 효율 향상을 목표로 한다.
- Cisco Cloud Control과의 연계를 통해 향후 확장 가능한 하이브리드 운영 모델을 지향한다.
